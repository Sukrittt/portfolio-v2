# Portfolio 2.0

Immersive 3D interactive portfolio built with Next.js 15, React Three Fiber, GSAP, Framer Motion, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **3D:** React Three Fiber + Three.js
- **Animation:** GSAP + Framer Motion
- **Scroll:** Lenis
- **Deploy:** Vercel

## Development

```bash
npm install
npm run dev
```
